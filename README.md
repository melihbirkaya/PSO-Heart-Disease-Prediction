# PSO Tabanlı Özellik Seçimi ile Kalp Hastalığı Tahmini

## Proje Hakkında

Bu çalışma Kırıkkale Üniversitesi Elektrik-Elektronik Mühendisliği bölümü kapsamında Optimizasyon Teknikleri dersi için hazırlanmıştır.

Çalışmada kalp hastalığı tahmini amacıyla Heart Disease veri seti kullanılmıştır. Logistic Regression sınıflandırıcısı uygulanmış ve özellik seçimi yaklaşımlarının model performansına etkisi incelenmiştir.

## Kullanılan Teknolojiler

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Veri Seti

Heart Disease Dataset
